Суть: Интерактивный Telegram-бот для параллельного тестирования и сравнения ответов от разных языковых моделей. Изначально настроен 
на GigaChat и YandexGPT, но архитектура легко масштабируется под любые LLM. Позволяет задавать одинаковые вопросы разным моделям, 
оценивать качество, стиль, скорость генерации и стоимость токенов, а в будущем — автоматически направлять запросы к оптимальной модели.
Стек: Python 3.11+ + aiogram 3.x + GigaChat API + YandexGPT API + LLM Router + Telegram Bot API + асинхронная обработка + логирование
🔹 Хотите такой же, но под вашу задачу?
Это не просто «демо-сравнивалка». Это готовый каркас для тестирования, A/B-анализа и интеллектуальной маршрутизации запросов. 
Позволяет бизнесу выбрать оптимальную модель по соотношению «цена/качество/скорость» и встроить её в рабочие процессы без 
переписывания бота с нуля.

🔄 Подо что можно изменить/адаптировать
Задача	                    Как работает	                                                    Для кого
**A/B тест промптов**	    Один запрос → разные системные инструкции → сравнение результатов	Маркетологи, продуктологи, AI-инженеры
**Маршрутизатор запросов**	Простые вопросы → дешёвая модель, сложные → мощная	                Корпоративные чат-боты, техподдержка
**QA-проверка ответов**	    Сравнение выводов нейросети с эталонной базой знаний	            Юристы, комплаенс, образовательные платформы
**Обучение / Демонстрация**	Наглядное сравнение «до/после» или «модель А vs модель Б»	        Тренеры, HR, стартапы, консалтинг
