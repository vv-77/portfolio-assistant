MultiModel Bot (GigaChat vs YandexGPT)

Интерактивный бот для сравнения ответов от разных больших языковых моделей. Выбираешь LLM — задаёшь вопрос — получаешь ответ. Можно задать один и тот же вопрос по очереди Яндексу и Гигачату и сравнить результаты.

Python Telegram Bot GigaChat YandexGPT LLM Comparison

📋 Описание

MultiModel Bot — это интерактивное демо-приложение, позволяющее сравнивать ответы от двух популярных российских больших языковых моделей: GigaChat и YandexGPT.

Ключевые возможности:

  • Выбор LLM перед отправкой вопроса (GigaChat или YandexGPT)
  • Отправка текстовых запросов и получение ответов от выбранной модели
  • Возможность задать один и тот же вопрос обеим моделям по очереди
  • Сравнение качества, стиля и содержания ответов
  • Простой и интуитивно понятный интерфейс

🤝 Поддерживаемые LLM

🟢
GigaChat
LLM от Сбера. Поддерживает русский язык, работает через API.
🟡
YandexGPT
LLM от Яндекса. Генерация текста, диалоговый режим, API.

По желанию можно добавить другие LLM (например, Qwen, Deepseek, Kimi, MiniMax).

💡 Сценарии использования

  • Сравнение моделей: Оценить, какая LLM лучше справляется с вашими задачами
  • Тестирование: Проверить качество ответов разных моделей на одинаковые запросы
  • Образование: Показать студентам различия в подходах разных LLM
  • Демо: Наглядно продемонстрировать возможности различных языковых моделей

🖼️ Скриншоты

🛠️ Технологии

  • Интерфейс: Telegram Bot API

  • Развёртывание: Гибкое (локально или на сервере)
