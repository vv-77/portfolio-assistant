Ням-Ням — Telegram-бот

Python-бот в Telegram с логикой рецептов, командами и сохранением состояния. Отличается простотой запуска и удобной структурой кода.

Что умеет

  • Принимает команды и запросы из Telegram.
  • Обрабатывает данные через aiogram и хранит состояние пользователя.
  • Поддерживает конфигурацию, логирование и работу с базой данных.

Технологии

  • Python
  • aiogram
  • Telegram Bot API
  • Асинхронный polling, FSM, logging
