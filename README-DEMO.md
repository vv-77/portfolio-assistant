# README-DEMO — MultiModel Bot (GigaChat vs YandexGPT)

## Чек-лист скриншотов для сдачи (5 пунктов)
1) `/start` и клавиатура с кнопками выбора модели
2) Переключение на **🟢 GigaChat** и ответ на вопрос
3) Переключение на **🔵 YandexGPT** и ответ на тот же вопрос
4) `/status` — отображение текущей модели
5) `/compare` — сравнение последних двух ответов (длина/стиль)

## Текст описания для преподавателя
Бот демонстрирует сравнение ответов двух LLM-провайдеров (GigaChat и YandexGPT) в одном чате.
Пользователь переключает активную модель кнопками, а команда `/compare` помогает быстро оценить различия
между последними ответами по длине и стилю.

## Инструкция
1) Заполнить `.env` на основе `.env.example`
2) Установить зависимости:

```powershell
pip install -r requirements.txt
```

3) Запустить:

```powershell
python main.py
```

## Примечания по ключам
- Секреты хранятся только в `.env` и не коммитятся.
- Для GigaChat нужны `GIGACHAT_CLIENT_ID` и `GIGACHAT_CLIENT_SECRET`.
- Для YandexGPT нужны `YANDEX_API_KEY` и `YANDEX_FOLDER_ID`.
