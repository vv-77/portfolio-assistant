HR AI Assistant B2B — Корпоративный RAG-консультант

Готовое B2B-решение для автоматизации внутренних запросов сотрудников. Отвечает по ТК РФ, отпускам, больничным, справкам и шаблонам документов, опираясь строго на вашу базу знаний. Архитектура универсальна: замена документов и системного промпта превращает бота в консультанта для любой корпоративной сферы.

Python Telegram Bot RAG LLM B2B

📋 Стек

Python 3.11+ + aiogram 3.x + FastAPI + ChromaDB/FAISS (RAG) + LLM API (Qwen/Deepseek) / Ollama (локальная) + SQLite/PostgreSQL + Docker + система лицензирования

🔹 Хотите такой же, но под вашу задачу?

Это не просто «HR-бот». Это каркас корпоративного AI-ассистента с тремя режимами развёртывания (Cloud / Hybrid / Boxed). Загружаете свои регламенты, пишете промпт — и система начинает консультировать сотрудников или клиентов без «галлюцинаций», с цитированием источников и соблюдением 152-ФЗ.

⚙️ Что ещё можно добавить

  • Интеграция с Bitrix24 / amoCRM / 1С для авто-передачи заявок и тикетов
  • Веб-виджет для интранета/корпоративного портала (работа без Telegram)
  • Аудит-логи: кто, когда и что спрашивал (для HR-аналитики и безопасности)
  • Голосовой ввод/вывод, мультиязычность, кэширование частых ответов
  • Автоматическое обновление базы знаний из Google Docs / Notion / SharePoint

От вас потребуется:

  • База знаний: регламенты, ТК/НК РФ, FAQ, шаблоны документов (PDF, Word, Excel, ссылки)
  • Цель: консультации сотрудников / обучение / сбор заявок / техподдержка
  • Предпочтительный режим: Cloud (быстро), Hybrid (анонимизация), Boxed (полный офлайн)
  • Требования к безопасности: 152-ФЗ, внутренние политики, ограничения на передачу данных в облако

📅 Сроки и бюджет

Сроки: 5–7 рабочих дней

Бюджет: от 25 000 ₽ (базовая настройка под вашу базу + тестирование)

Не входит: Аренда сервера, оплата LLM-токенов, наполнение базы с нуля (только структура и парсинг), кастомный дизайн веб-интерфейса, рекламное продвижение

💳 Оплата — как удобно

  • Разово: 30–50% предоплата → демо на моём сервере → сдача → остаток
  • Подписка: 15% от стоимости/мес (мин. 5 000 ₽) — бот работает сразу, выкупаете когда готовы
  • Исходный код + инструкция по развёртыванию и администрированию — ваши после полной оплаты

🔹 Поддержка и правки

  • ✅ На подписке: правки, баги, доработки, обновление базы знаний — за мой счёт, приоритет в очереди
  • ✅ После выкупа: 14 дней гарантии на баги с моей стороны → потом ремонт по мере свободного времени (символическая плата)
  • 🚀 Вне очереди? Подключите сопровождение от 1 500 ₽/мес — откладываю всё, чиню ваш проект сразу
