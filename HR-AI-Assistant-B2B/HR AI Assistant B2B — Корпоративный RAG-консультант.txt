Суть: Готовое B2B-решение для автоматизации внутренних запросов сотрудников. Отвечает по ТК РФ, отпускам, больничным, 
справкам и шаблонам документов, опираясь строго на вашу базу знаний. Архитектура универсальна: замена документов и системного 
промпта превращает бота в консультанта для любой корпоративной сферы.
Стек: Python 3.11+ + aiogram 3.x + FastAPI + ChromaDB/FAISS (RAG) + LLM API (Qwen/Deepseek) / Ollama (локальная) + 
SQLite/PostgreSQL + Docker + система лицензирования
🔹 Хотите такой же, но под вашу задачу?
Это не просто «HR-бот». Это каркас корпоративного AI-ассистента с тремя режимами развёртывания (Cloud / Hybrid / Boxed). 
Загружаете свои регламенты, пишете промпт — и система начинает консультировать сотрудников или клиентов без «галлюцинаций», 
с цитированием источников и соблюдением 152-ФЗ.
