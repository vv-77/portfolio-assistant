Суть: Готовое веб‑приложение для быстрого анализа табличных данных. Загружаете Excel/CSV/PDF → система автоматически строит 
графики, считает статистику и генерирует текстовый разбор через LLM. Экспорт в PDF работает полностью на стороне клиента — без 
нагрузки на сервер. Архитектура открыта: меняйте провайдеров, подключайте шаблоны отчётов и масштабируйте под бизнес‑задачи.
Стек: Python 3.11+ + FastAPI + JavaScript/TypeScript + Plotly + html2canvas/jsPDF + LLM API (Yandex/GigaChat/Gemini/Deepseek) + Docker + конфиги (.env/.yaml)
🔹 Хотите такой же, но под вашу задачу?
Это не просто «анализатор файлов». Это каркас корпоративного аналитического инструмента с тремя режимами развёртывания (Cloud / Hybrid / Boxed). 
Подключаете свои данные, настраиваете шаблоны отчётов — и получаете автоматизированного аналитика, который работает без «чёрного ящика», 
с прозрачной логикой расчётов и полной изоляцией чувствительных данных (client‑side экспорт, локальные чанки, контроль MAX_FILE_SIZE_MB).
