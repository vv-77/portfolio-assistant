DataInsight PRO — Веб‑приложение для авто‑анализа данных и AI‑отчётов

Веб-приложение для автоматического анализа данных и генерации AI-отчётов. Загружаете данные — получаете готовые аналитические отчёты с визуализацией и выводами.

Python FastAPI Data Analysis AI Reports Visualization

📋 Стек

Python 3.11+ + FastAPI + pandas/openpyxl + matplotlib/plotly + LLM API (Qwen/Deepseek/Gemini) + HTML5/CSS3/JS + адаптивная вёрстка

🔹 Основные возможности

  • Загрузка данных в форматах Excel, CSV, JSON
  • Автоматический анализ структуры и качества данных
  • Генерация аналитических отчётов с помощью AI
  • Визуализация данных: графики, диаграммы, сводки
  • Экспорт отчётов в PDF, Excel, HTML

От вас потребуется:

  • Данные: примеры файлов (Excel, CSV) или доступ к источнику
  • Цель анализа: какие метрики и выводы важны
  • Формат отчёта: PDF / Excel / HTML / веб-дашборд
  • AI-модель: предпочтения по LLM или используем по умолчанию

📅 Сроки и бюджет

Сроки: 5–7 рабочих дней

Бюджет: от 15 000 ₽ (базовая версия с анализом + отчёт)

Не входит: Аренда сервера, оплата AI-токенов, сложная интеграция с legacy-системами, дизайн с нуля

💳 Оплата — как удобно

  • Разово: 30–50% предоплата → демо на моём сервере → сдача → остаток
  • Подписка: 15% от стоимости/мес (мин. 5 000 ₽) — инструмент работает сразу, выкупаете когда готовы
  • Исходный код + инструкция: по развёртыванию — ваши после полной оплаты

🔹 Поддержка и правки

  • ✅ На подписке: правки, баги, доработки, обновление под новые данные — за мой счёт, приоритет в очереди
  • ✅ После выкупа: 14 дней гарантии на баги с моей стороны → потом ремонт по мере свободного времени (символическая плата)
  • 🚀 Вне очереди? Подключите сопровождение от 1 500 ₽/мес — откладываю всё, чиню ваш проект сразу
