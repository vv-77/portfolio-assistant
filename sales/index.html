HR AI Assistant B2B (Sales Kit)

Корпоративный HR-ассистент с тремя тарифами (Cloud/Hybrid/Boxed), RAG-поиском по базе знаний, поддержкой Telegram и Web API. Полная документация, скрипты онбординга и лицензирования.

Python Telegram Bot RAG LLM B2B

📋 Описание

Проект представляет собой готовое решение для автоматизации HR-запросов сотрудников. Бот отвечает на вопросы по ТК РФ, отпускам, больничным, справкам и шаблонам документов.

Ключевые особенности:

  • Три тарифа: Cloud (облачный LLM), Hybrid (локальная БД + облачный LLM), Boxed (полный офлайн с Ollama)
  • RAG-поиск по загруженной базе знаний компании
  • Поддержка Telegram и Web API
  • Анонимизация персональных данных в Hybrid-режиме
  • Система лицензирования для Boxed-тарифа

ТарифРежимДля когоЧто входит

📁 Структура проекта

sales/ — единый файл документации (SALES-KIT-ALL.md) с полной информацией:

  • TARIFFS.md — описание тарифов
  • UPGRADE-GUIDE.md — руководство по миграции
  • support/ — FAQ, troubleshooting, escalation flow
  • billing/ — шаблоны счетов, напоминаний, генератор лицензий
  • tariffs/ — скрипты онбординга для каждого тарифа
  • scripts/ — утилиты диагностики и миграции
