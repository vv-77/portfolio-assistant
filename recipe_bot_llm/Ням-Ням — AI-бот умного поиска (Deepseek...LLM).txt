Суть: Telegram-бот, который принимает текстовые запросы и фото, возвращает точные ответы на основе загруженной базы. 
Изначально настроен под рецепты и мультимодальный поиск ингредиентов, но архитектура универсальна: работает как 
интеллектуальный ассистент для любой предметной области.
Стек: Python 3.11+ + aiogram 3.x + Deepseek API (или другая LLM) + Telegram Bot API + асинхронная обработка + логирование + конфиг-файл
🔹 Хотите такой же, но под вашу задачу?
Это не просто «бот рецептов». Это гибкая платформа умного поиска, которая заменяет ручной перебор документов и выдаёт готовые 
подборки, расчёты или инструкции. Архитектура легко адаптируется под автозапчасти, юриспруденцию, HR, медицину, ритейл.
🔹 Подо что можно изменить
• Автозапчасти: поиск по фото детали, проверка аналогов, наличие, цена
• Юристы / Бухгалтеры: ответы по кодексам, формам, срокам сдачи отчётности
• HR / Обучение: онбординг сотрудников, ответы по регламентам, сбор заявок
• Фудтех / Рестораны: подбор блюд, подсчёт КБЖУ, экспорт списка покупок
🔹 Что ещё можно добавить
• Замена модели на любую по вашему выбору: Qwen, Gemini, YandexGPT или локальную Ollama
• Поиск по фото (распознавание объектов/ингредиентов/документов)
• Подсчёт калорийности, составление меню/планов, интеграция с доставкой продуктов
• Избранное, история запросов, рассылка новинок/акций
От вас потребуется:
• Источники данных: база рецептов, каталог, PDF-книги, сайт, 1С, CRM
• Логика работы: поиск по тексту / фото / категориям / случайная подборка
• Интеграции (опционально): корзина, избранное, доставка, уведомления
Сроки: 3–5 рабочих дней
Бюджет: от 10 000 ₽ (базовая версия с поиском по тексту + фото)
