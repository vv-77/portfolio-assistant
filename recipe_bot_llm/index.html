Ням-Ням — Бот рецептов (Deepseek)

Telegram-бот, который предлагает рецепты, ищет по фото и работает через Deepseek API. Для портфолио важно не только код, но и понятное описание, навигация и демонстрация.

Что делает проект

  • Принимает текстовые запросы и возвращает рецепты.
  • Работает в Telegram и использует Deepseek для LLM-подсказок.
  • Есть поиск изображений и поддержка мультимодальных сценариев.

Технологии

  • Python
  • Telegram Bot API
  • Deepseek API
  • Асинхронный aiogram, логирование и файл конфигурации
