Остров Окон — AI-консультант

Голосовой / текстовый консультант для оконного бизнеса, собранный в конструкторе Savvi и настроенный на работу с LLM Qwen. Задача — дать понятный ответ клиенту и предложить выгодное решение.

Что реализовано

  • Интерактивная консультация по выбору окон и стеклопакетов.
  • Сценарии диалога для коммерческих предложений.
  • Интеграция с Qwen для генерации текста и обработки запросов.

Технологии

  • Savvi — конструктор бота
  • Qwen LLM
  • AI-консультирование и сценарии продаж
  • UX-прототип для бизнеса
