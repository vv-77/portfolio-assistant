Суть: Текстовый AI-консультант, изначально собранный в конструкторе Savvi под оконный бизнес. Отвечает на вопросы клиентов, 
подбирает решения, генерирует коммерческие предложения. Сейчас работает на базе Qwen LLM с готовыми сценариями продаж. 
Архитектура легко переносится на чистый код и масштабируется под любую нишу.
Стек (сейчас): Savvi + Qwen LLM + UX-прототип
Стек (на Python): aiogram 3.x / FastAPI + LLM API (Qwen/Deepseek/Gemini/локальная) + ChromaDB/FAISS (RAG) + SQLite/PostgreSQL + Docker
🔹 Хотите такой же, но не в конструкторе, а на Python?
Конструкторы удобны для прототипа, но ограничивают логику, интеграции и контроль данных. Перенесу на чистый код: полная 
кастомизация сценариев, прямая работа с вашей базой/CRM, отсутствие ежемесячных плат платформе, гибкая настройка под ваши процессы.

🔹 Что ещё можно добавить
• Интеграция с 1С / Bitrix24 / amoCRM для автоматической передачи заявок
• Анонимизация данных перед отправкой в LLM (соответствие 152-ФЗ)
• Голосовой ввод/вывод, мультиязычность, кэширование ответов для экономии токенов
• Веб-виджет для установки на сайт (чат-консультант без Telegram)
От вас потребуется:
• База знаний: прайс, каталог, ответы на частые вопросы, регламенты (PDF, Excel, сайт, текст)
• Системный промпт: роль, тон ответов, правила отказа от «галлюцинаций»
• Цель: консультация → расчёт → сбор заявки → передача менеджеру
Сроки: 3–5 рабочих дней
Бюджет: от 12 000 ₽ (базовая версия под вашу базу)
Не входит: Аренда сервера, оплата LLM-токенов, парсинг закрытых данных без разрешения, дизайн лендинга
🔹 Оплата — как удобно
• Разово: 30–50% предоплата → демо на моём сервере → сдача → остаток
• Подписка: 15% от стоимости/мес (мин. 5 000 ₽) — бот работает сразу, выкупаете когда готовы
• Исходный код + инструкция по запуску — ваши после полной оплаты
🔹 Поддержка и правки
✅ На подписке: правки, баги, доработки — за мой счёт, приоритет в очереди
✅ После выкупа: 14 дней гарантии на баги с моей стороны → потом ремонт по мере свободного времени (символическая плата)
